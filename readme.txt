A plugin to disable the shopping feature in WooCommerce.

== Description ==

This plugin disables the shopping feature in WooCommerce. It's useful for businesses that want to display their products without allowing online purchases.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nws-disable-woocommerce-shopping` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

Yes, this plugin requires WooCommerce.
